# 🏥 AI Hospital Assistant

An AI-powered Hospital Assistant that helps patients interact with a chatbot, register themselves, and book appointments with doctors. The application supports voice input, AI-based responses, patient registration, and appointment management.

---

## 🚀 Live Demo

🌐 Frontend: https://hospital-voice-assistnce.vercel.app/

---

## ✨ Features

- 🤖 AI Hospital Chat Assistant
- 🎤 Voice-to-Text support
- 💬 Interactive chatbot conversation
- 👨‍⚕️ Doctor information page
- 📝 Patient Registration
- 📅 Appointment Booking
- 🌙 Dark / Light Theme
- 📱 Responsive Design
- ☁️ MongoDB Atlas Database
- 🚀 Deployed using Vercel & Render

---

## 🛠️ Tech Stack
